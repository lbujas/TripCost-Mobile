# Noto Sans PDF fonts (SIL Open Font License 1.1)

Bundled for packing list PDF export:

- `NotoSans-Regular.ttf`
- `NotoSans-Bold.ttf`

To (re)download:

```bash
dart run tool/download_pdf_fonts.dart
```

Source: [googlefonts/noto-fonts](https://github.com/googlefonts/noto-fonts)

PDF export fails with a clear error if these files are missing.
